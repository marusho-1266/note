# ディレクトリ構成

以下のディレクトリ構造に従って実装を行ってください：

```
/
├── draft/                        # 素案記事
├── editing/                      # 校正中記事
├── article/                      # 本稿記事保管場所
├── Contribution/                 # 寄稿済記事
├── inbox/                        # アイディア
├── knowledge/                    # 雑多メモ
├── Clippings/                    # 情報収集
└── .cursor/                      # Cursor設定
```
